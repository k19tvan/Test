# Problem 05 - Sampling Offsets Initialization

## Description
Multi-scale deformable attention learns relative offsets from reference points to actual sampling locations. These offsets are generated by a linear layer, and proper initialization is crucial. We initialize the bias with a radial grid pattern that distributes sampling points around the reference in a learnable manner.

## Input Format
- **num_heads** (int): Number of attention heads
- **num_levels** (int): Number of feature pyramid levels
- **num_points_list** (list): Points per level, e.g., [4,4,4]
- **embed_dim** (int): Embedding dimension

## Output Format
- Linear layer module with:
  - Input size: embed_dim
  - Output size: total_points * 2 (x,y offsets for each point)
  - Bias initialized with radial grid pattern
  - Weights initialized to zero

## Constraints
- Output: (B, L, total_points * 2)
- Bias should create circular grid pattern around each reference point
- Weights initialized to constant 0
- Device-agnostic

## Example
num_heads=8, num_levels=3, num_points_list=[4,4,4], embed_dim=256
Linear layer input: 256, output: 8*12*2 = 192

## Hints
- Total points = sum(num_points_list) * num_heads
- Create radial grid: angle θ from 0 to 2π
- For each head and level: (r*cos(θ), r*sin(θ)) where r = scale factor
- Flatten and assign to bias
- Set weight to 0 (requires grad)
