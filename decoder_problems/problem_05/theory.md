# Problem 05 Theory - Sampling Offsets

## Theory
Deformable attention seeks optimal sampling locations for each query point at each feature level. These locations are computed as:
$$\text{sampling\_location} = \text{reference\_point} + \text{offset}$$

The offset is generated by a learned linear layer: $\text{offset} = W \cdot \text{query} + \text{bias}$

Proper bias initialization matters because it determines the initial "spread" of sampling points. A radial grid initialization ensures that initially, offsets are distributed uniformly around the reference point.

### Radial Grid Formula
For each of num_points per level, distribute on circle:
$$\theta_i = \frac{2\pi \cdot i}{n_{\text{points}}}$$
$$\text{bias}(i) = (\text{radius} \cdot \cos(\theta_i), \text{radius} \cdot \sin(\theta_i))$$

Where radius increases with level: $\text{radius} = (\text{scale})^{\text{level}}$

## Key Equations
$$\text{offset} = W \text{query} + b_{\text{init}}$$
$$\text{sampling\_location} = \text{ref} + \text{offset}$$
