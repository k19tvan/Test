# Problem 07 Questions

## Multiple Choice

1. **In an MLP with 3 layers, how many activation functions are applied?**
   - A) 0
   - B) 2
   - C) 3
   - D) 4
   - Answer: B

2. **Why is no activation applied to the output layer?**
   - A) To reduce computation
   - B) To allow unconstrained output values
   - C) To save memory
   - D) For gradient flow optimization
   - Answer: B

3. **What does the hidden_dim parameter control?**
   - A) Number of layers
   - B) Dimension of intermediate hidden layers
   - C) Input feature dimension
   - D) Batch processing dimension
   - Answer: B

4. **For MLP(256, 512, 128, 3), how many linear transformations occur?**
   - A) 2
   - B) 3
   - C) 4
   - D) 256
   - Answer: B

5. **What is ReLU's output for negative inputs?**
   - A) Input value unchanged
   - B) 0
   - C) -1
   - D) 1
   - Answer: B

## Answer Key
1.B 2.B 3.B 4.B 5.B
